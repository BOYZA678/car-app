เชื่อมต่อ Google Sheets
Apps Script Web App URL
วาง URL ที่ได้จากการ Deploy Google Apps Script เพื่อให้แอปบันทึกข้อมูลลง Spreadsheet ของคุณโดยตรง
วิธีตั้งค่า (ทำครั้งเดียว)
1
สร้าง Google Spreadsheet
ไปที่ sheets.google.com → สร้าง Spreadsheet ใหม่ → สร้างชีต: fuel gas ev service garage
2
เปิด Apps Script
ใน Spreadsheet → เมนู Extensions → Apps Script → วาง Code ที่ได้รับด้านล่าง
3
Deploy เป็น Web App
Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone
4
คัดลอก URL มาวาง
คัดลอก Web app URL จากหน้า Deploy มาวางในช่องด้านบน แล้วกดบันทึก
5
เริ่มใช้งานได้เลย!
ทุกรายการที่บันทึกในแอปจะถูกส่งไปยัง Google Sheets ของคุณทันที
ซิงค์ข้อมูลจาก Google Sheets
โหลดข้อมูลรถ ประวัติ และรอบบำรุงรักษาจาก Sheets มาเขียนทับข้อมูลในเครื่อง ใช้เมื่อเปิดในอุปกรณ์ใหม่หรือข้อมูลไม่ตรงกัน
รีเซ็ตข้อมูลในเครื่อง
ลบข้อมูลรถ ประวัติ และรอบบำรุงรักษาทั้งหมดออกจาก browser นี้ ใช้เมื่อต้องการเริ่มต้นใหม่หรือแก้ปัญหาข้อมูลค้าง
